Who am I?

I'm Dylan Scheltens.

I am a hobbyist programmer mainly involved in Minecraft-related projects.
I've been in the Minecraft modification & plugin development community since early 2012.
I started off as a modification developer working on mods such as Cheating Essentials and Arcticraft. And over time moved onto plugin development.

My first few plugins included hub plugins, minigames and bigger titles such as UHC core plugins.
I then started getting more interested in networking, including BungeeCord; which I have made a few plugins for, including a server manager.
Recently I've been working on a Spigot Framework used to improve content-production efficiency, and overall user experience.


Although I mainly work on Minecraft-related projects. I do have a history in other programming tasks. Including game development.
I used to work on a game called Seed Of Andromeda as an intern programmer, mainly having done basic GUI work. The game is a physics-based voxel game.
More info about this can be found here.

Besides that, I also have experience modding games such as Garry's Mod and Roblox.

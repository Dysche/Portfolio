What is Cantaloupe?

Cantaloupe is a Spigot-based Framework made entirely from scratch. It's useful to people who want to create content-rich plugins easily,
without the hassle of having to write tons of classes to achieve a simple goal.

The Framework contains tons of classes, utilities and methods made to improve the overall experience of creating plugins for both the new and experienced developer.
This includes everything from Holograms and Statues, to a built-in ready-to-use Audio Server to enhance the player experience.


About the Audio Server

The Audio Server is a ready-to-use utility used to create more interactive experiences for the player.
This can include anything from background music to sound effects.

The server comes with a ready-to-use client which can be read about here.
